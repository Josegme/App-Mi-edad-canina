# Mi Edad Canina - Proyecto Android

Este es un proyecto de ejemplo desarrollado para la materia de **Programación Móvil** de la carrera de **Gastronomía y Programación de Interfaces (GPI)** en el **INCADE**.

## Descripción del Proyecto

`Mi Edad Canina` es una aplicación sencilla para Android que permite a los usuarios calcular cuál sería su edad si fueran un perro. El objetivo principal de este proyecto es demostrar los conceptos fundamentales del desarrollo de aplicaciones Android, incluyendo:

-   Manejo de la interfaz de usuario (UI) con XML.
-   Interacción del usuario a través de botones y campos de texto.
-   Implementación de lógica de negocio básica en Java.
-   Uso de `View Binding` para conectar de forma segura los componentes de la UI con el código.

### Funcionalidades

-   Un campo de texto (`EditText`) para que el usuario ingrese su edad.
-   Un botón (`Button`) para iniciar el cálculo.
-   Un campo de texto (`TextView`) para mostrar el resultado del cálculo.
-   Validación para mostrar un mensaje de error si no se ingresa una edad.

## Tecnologías Utilizadas

-   **Lenguaje:** Java
-   **IDE:** Android Studio
-   **Componentes de Android Jetpack:**
    -   `View Binding`: Para eliminar la necesidad de `findViewById` y crear interacciones más seguras con las vistas.
-   **SDK:** Android SDK
